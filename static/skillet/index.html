Skillet

Welcome to Skillet!

Skillet is a personal project made to help me with practicing various skills.

The most obvious question it helps me solve is: How often am I doing something right or wrong? I used to answer it with a stopwatch and a piece of paper, but it was unnecessarily tedious. I thought about using my flashcard program to help, but it would spread out the review a bit too much for my taste when I'm trying to drill some new skill.

If you like this project, and want to see more from me, visit my website!

Controls

p: Success (you did the thing).

f: Failure (you didn't do the thing).

spacebar: Start the timer. If timer is already started, stop the timer and succeed the run.

a: Abort the timer without failing (use when something out of your control goes wrong).

u: Undo the last pass/fail event.
